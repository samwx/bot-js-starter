# Bot js starter
Simple bot with javascript (ES6 + Babel) using [BLiP Platform](http://portal.blip.ai).

## Usage
* On `index.js`, setup `identifier` and `accessKey` vars. More infos in [BLiP docs](https://portal.blip.ai/#/docs/sdks/js/instantiating).
* `npm install`
* `npm start`

## Requirements
* `node v7.6 or higher`
